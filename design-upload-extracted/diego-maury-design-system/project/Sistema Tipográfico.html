Cómo se ven juntas

Diseño y dirijo programas que mueven la aguja.

Montserrat lleva el peso visual en los titulares y la interfaz. Bitter en itálica aporta el tono humano y editorial en subtítulos y citas. Space Mono marca las etiquetas y los datos con precisión.

04 — Reglas

Buenas prácticas

Para que el sistema se mantenga consistente en cualquier formato.

Sí

  • Montserrat 800/900 en MAYÚSCULAS para los titulares principales.
  • Bitter siempre en itálica cuando es subtítulo o acento editorial.
  • Space Mono solo en mayúsculas y con mucho espaciado entre letras.
  • Morado para textos principales; naranja reservado para etiquetas y acentos.

No

  • No introducir otras fuentes (Satoshi, Arial, Inter, etc.).
  • No usar Space Mono en párrafos largos ni en minúsculas.
  • No poner Bitter itálica en bloques largos de lectura.
  • No usar naranja para cuerpo de texto: pierde legibilidad.
05 — Implementación

Para pegar en cualquier proyecto

Carga las fuentes y usa estas variables. Funciona igual en web, y los nombres existen tal cual en Canva, Figma y Office.

<!-- 1. Cargar en el <head> -->
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;700;800;900&family=Bitter:ital,wght@0,400;1,500&family=Space+Mono:wght@400;700&display=swap" rel="stylesheet">

/* 2. Tokens CSS */
--font-display: 'Montserrat', sans-serif; /* titulares + UI */
--font-serif  : 'Bitter', serif; /* subtítulos (italic) */
--font-mono   : 'Space Mono', monospace; /* etiquetas + datos */

/* 3. Color de marca */
--purple: #4b2a87;  --orange: #f0592a;
