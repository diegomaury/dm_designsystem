Hagamos que las cosas pasen

Cómo instalarla

  1. Gmail: Ajustes → Ver toda la configuración → Firma → selecciona el bloque de arriba (desde el isotipo hasta "Hagamos que las cosas pasen"), cópialo y pégalo en el editor de firma.
  2. Outlook: Archivo → Opciones → Correo → Firmas → pega el bloque copiado.
  3. Apple Mail: Preferencias → Firmas → pega y desactiva "Usar siempre la fuente predeterminada".
  4. El isotipo debe estar en una URL pública (p. ej. diegomaury.mx/isotipo.png) para que se vea en el correo enviado. Mientras tanto usa el copiado directo.
